.. _roadInstallation:

Installation
=====

#. Create RoadSegments.
#. Configure paths.
#. Create required PedestrianNode by PedestrianNodeCreator.
#. Configure semaphores.
#. Create TrafficAreas (optional).
#. Create PublicRoutes (optional).
#. Open RoadParent.
#. Press Reset.
#. Press Connect.
#. Press Bake.
